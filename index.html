    Welcome to the diamond registry API! You can use the diamond registry to get and manage packages for diamond.

    API Root: registry.diamond.js.org

    All requests should use SSL.

    Versions

    All requests should prefix requests with a version. E.g. registry.diamond.js.org/v1/foo

    Name Out of Service?
    v1 No

    Authentication

    Your Authorization header should look like this:

    Authorization: Basic eW91IHRvb2sgdGhlIHRpbWUgdG8gcmVhZCB0aGlzPyBnZw==

    Authentication is done via HTTP Basic so you should always use SSL.

    User

    User Object

    {
  "username": "hackzzila",
  "email": "admin@hackzzila.com",
  "createdAt": "2017-05-01T18:58:47.263Z",
  "packages": [
    "registry"
  ]
}
    Key Type Description
    username string The user’s username
    email string The user’s email
    createdAt date When the user was created
    packages Array<string> All packages the user is an author of. May not be present on all user objects

    Get Current Logged in User

    GET /user HTTP/1.1
Host: registry.diamond.js.org
Authorization: Basic eW91IHRvb2sgdGhlIHRpbWUgdG8gcmVhZCB0aGlzPyBnZw==

    The above request returns JSON structured like this:

    {
  "username": "hackzzila",
  "email": "admin@hackzzila.com",
  "createdAt": "2017-05-01T18:58:47.263Z",
  "packages": [
    "registry"
  ]
}
    Requires authentication

    This endpoint returns the current logged in user.

    HTTP Request

    GET /user

    Reponse Format

    A User Object is returned including the packages key.

    Key Type Description
    name string The package name
    description string The latest version’s description
    createdAt date When the package was created
    editedAt date When the package was last updated
    tags Object<string, SemVer> An object mapping tag names to version numbers. May be empty on some package objects
    authors Array<User> An array of all package authors. May be empty on some package objects
    versions Object<SemVer, Version> An object mapping version numbers to version objects. May be empty or not full on some package objects

    Dist Object

    {

}
    Key Type Description
    shasum sha256sum The sha256sum for the tarball
    url string A link to where to download the package

    This endpoint gets a package by name.

    HTTP Request

    GET /package/:name

    URL Parameters

    Key Type Description
    name string The name of the package you want to get

    Reponse Format

    A Package Object is returned including all keys.

    Download Package

    GET /package/foo/0.1.0 HTTP/1.1
Host: registry.diamond.js.org

    The above request returns a gzipped tarball (.tar.gz)

    This endpoint gets package contents.

    HTTP Request

    GET /package/:name/:version

    URL Parameters

    Key Type Description
    name string The name of the package you want to download
    version SemVer | string The version or tag that you want to download

    Reponse Format

    A gzipped tarbal (.tar.gz) file is returned.
